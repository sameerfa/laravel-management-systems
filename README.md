## Library Management System

An integrated library system, also known as a library management system, is an enterprise resource planning system for a library, used to track items owned, orders made, bills paid, and patrons who have borrowed

![alt text](https://raw.githubusercontent.com/sameerfa/laravel-management-systems/Library/public/library.png)

## Deployment

Set DB Credentials on .env file, you can copy the env example file and edit.

```php
composer install
```

```php
php artisan migrate --seed
```

```php
php artisan key:generate
```

```php
php artisan storage:link
```
